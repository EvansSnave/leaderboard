### Prerequisites

In order to run this project you need: have 'npm' package manger. Install webpack and webpack server. 

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/ahmedeid6842/Portfolio.git
```
